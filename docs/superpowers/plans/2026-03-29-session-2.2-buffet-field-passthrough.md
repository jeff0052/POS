# Session 2.2 — Buffet Field Passthrough Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add buffet fields (isBuffetIncluded, buffetSurchargeCents, buffetInclusionType) to order item entities, copy them during order submission, and adjust lineTotal based on buffet pricing rules.

**Architecture:** Pure field passthrough — add 3 columns (already in DB via V074) to 2 JPA entities, update the `toSubmittedItem()` copy method with lineTotal adjustment logic, fix `mergeQrItems()` to preserve buffet fields, and add buffet fields to 2 DTOs for frontend visibility.

**Tech Stack:** Java 17, Spring Boot 3.3.3, JPA/Hibernate, MySQL 8.4

**Spec References:**
- `docs/superpowers/specs/final-executable-spec.md` D6 (buffet 价格持久化)
- `docs/84-implementation-plan-and-roadmap.md` Session 2.2 + Remarks

**D6 Spec vs DDL discrepancy (resolved):** D6 originally specified `buffet_package_id` as a field to copy on order items. V074 DDL replaced this with `buffet_inclusion_type` (VARCHAR: INCLUDED/SURCHARGE/EXCLUDED). No `buffet_package_id` column exists on order item tables — it lives on `table_sessions` (session-level, via V067). This plan follows V074 DDL. Confirmed with user during brainstorm.

**`buffetSurchargeCents` semantics:** This is a **per-unit** surcharge value, consistent with the existing codebase convention where `lineTotalCents = unitPrice * quantity`. When computing lineTotal for surcharge items: `lineTotalCents = buffetSurchargeCents * quantity`.

### Task 3: Update toSubmittedItem() with buffet field copy + lineTotal adjustment

**Files:**
- Modify: `/Users/ontanetwork/Documents/Codex/pos-backend/src/main/java/com/developer/pos/v2/order/application/service/ActiveTableOrderApplicationService.java:460-469`

- [ ] **Step 1: Add buffet field copy and lineTotal logic to `toSubmittedItem()`**

Replace current method (lines 460-469) with:

```java
private SubmittedOrderItemEntity toSubmittedItem(ActiveTableOrderItemEntity item) {
    SubmittedOrderItemEntity next = new SubmittedOrderItemEntity();
    next.setSkuId(item.getSkuId());
    next.setSkuCodeSnapshot(item.getSkuCodeSnapshot());
    next.setSkuNameSnapshot(item.getSkuNameSnapshot());
    next.setQuantity(item.getQuantity());
    next.setUnitPriceSnapshotCents(item.getUnitPriceSnapshotCents());
    next.setItemRemark(item.getItemRemark());

    // Buffet field passthrough (D6)
    next.setBuffetIncluded(item.isBuffetIncluded());
    next.setBuffetSurchargeCents(item.getBuffetSurchargeCents());
    next.setBuffetInclusionType(item.getBuffetInclusionType());

    // Buffet lineTotal adjustment (D6):
    //   INCLUDED (surcharge=0) → free, lineTotal=0
    //   SURCHARGE (surcharge>0) → lineTotal=surcharge * quantity (per-unit surcharge)
    //   EXCLUDED or non-buffet  → lineTotal=original price
    if (item.isBuffetIncluded()) {
        if (item.getBuffetSurchargeCents() > 0) {
            next.setLineTotalCents(item.getBuffetSurchargeCents() * item.getQuantity());
        } else {
            next.setLineTotalCents(0);
        }
    } else {
        next.setLineTotalCents(item.getLineTotalCents());
    }

    return next;
}
```
