# PC Admin

Merchant admin skeleton for the POS project.

## Included

- React + Vite + TypeScript project files
- Ant Design based admin layout
- Local auth context and protected routes
- Switchable service layer with mock fallback
- Login page
- Dashboard page
- Products page
- Categories page
- Orders page
- Reports page
- Order detail drawer
- Refund records page
- Product/category form skeletons
- Local draft editing for product/category forms
- Refund detail drawer

## Next recommended steps

1. Install dependencies with `npm install`
2. Change `USE_MOCK_API` in `src/api/config.ts`
3. Point service URLs to the real backend
4. Add persistent query/cache layer
5. Replace local draft submit with real create-edit-submit APIs
